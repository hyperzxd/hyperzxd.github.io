使用Github-Hexo搭建Blog

  • Setup node.js

brew install node

  • Setup hexo

npm install -g hexo

  • Setting hexo

$ hexo init
$ npm install
$ npm install hexo-deployer-git –save
edit config.yml
url: http://your_name.github.io/
deploy:
type: git
repository: git@github.com:your_name/your_name.github.io
branch: master

  • Setting SSH

$ ssh-keygen -t rsa -C ‘your_email’
$ ssh-agent -s
$ ssh-add ~/.ssh/id_rsa
Copy ~/.ssh/id_rsa.pub to github’s SSH

  • Use Hexo

$ hexo n ‘hello world’
$ hexo g
$ hexo s Testing
$ hexo d
